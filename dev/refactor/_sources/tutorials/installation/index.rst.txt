Installation
===================

pyspj is currently hosted on PyPI. It required python >= 3.6.

You can simply install pyspj from PyPI with the following command:

.. code:: shell

    pip install pyspj

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/hansbug/pyspj.git@main

script:
